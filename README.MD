## Setup project: 
1. Install redis package
   (Ubuntu:)
   $ apt-get install redis-server 

2. Use docker compose:
$ docker-compose up

3. Open http://0.0.0.0:5000/receive-sse-sensor-data
4. Open http://0.0.0.0:5000/
